Задание 2.2
Поиск числового ключа
Постановка задачи:
Входные данные:
1) текстовый файл, каждая из строк которого содержит 4 поля в табличном виде (с первого по третье поле - см. задание 2.1, четвертое - номер заявки (число). Все данные во входном файле корректные (проверять на корректность не нужно). Количество строк входного файла - 1000000.
2) целое число n - количество строк входного файла для обработки (10 ≤ n ≤ 1000000).
3) ключ поиска.
Выходные данные:
1) номер строки (или список номеров строк) (см. свой вариант задания) входного файла, содержащей (содержащих) заданный ключ поиска.
2) количество шагов поиска каждым из алгоритмов (см. свой вариант задания) (без учета сортировки любым алгоритмом задания 1.3).
Содержание отчета:
1. Титульный лист
2. Неформальная постановка задачи
3. Описание алгоритмов поиска (не идея, а ваша реализация алгоритма)
4. Текст программы
5. Тесты
